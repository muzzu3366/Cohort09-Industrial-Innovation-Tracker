# Cohort09-Industrial-Innovation-Tracker
The Industrial Innovation Tracker project analyzes and predicts a country’s innovation output using research and development indicators. Patent applications are used as a measure of innovation, and predictions are made based on R&D expenditure (% of GDP) and the number of researchers per million people. The project involves data preprocessing, exploratory data analysis, multicollinearity checking using VIF, and model comparison between Linear Regression and Random Forest. The results show that innovation performance is influenced by both financial investment and skilled human capital, with Random Forest providing stronger predictive accuracy.
